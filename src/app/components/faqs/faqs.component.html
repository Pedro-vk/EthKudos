What is gas? add

Gas is a shorthand used to describe the cost of powering a transaction or contract in Ethereum, which is the blockchain network that Ethereum is built on.

Because blockchain is decentralized, every transaction is distributed through multiple computers, not a central server. This ensures each organisation and polling is secure or one-of-a-kind. However, it also takes more computational power, which is covered by the cost of gas. Ethereum typically communicates this cost in gwei (1 million gwei = 1 ether/ETH).

Understanding EthKudos

What is EthKudos? add

EthKudos is a gratitudes/kudos system that provides to companies, organisations or teams to boost the cooperation between the coworkers.

EthKudos allows to create a new organisation over Ethereum, this organisation is owned by the creator. Ethereum is the technology that ensures that the owner of the organisation and the members are who can interact with this organisation.

Hold your horses, describe EthKudos in words that make sense to me add

You will be able to create an organisation, add members and create polling events. In a polling, the members are able to vote for other members. At the end of a polling, this members will receive the kudos sent by other members.

What is an organisation on EthKudos? add

An organisation, technically, is a Smart contract that is called ERC20 (or Token ERC20) with more features. It means that the members will be able to see his/here balance of kudos on MetaMask or other Ethereum client.

Each organisation have its own name, symbol and number of decimals. It is the necessary information to create a Token ERC20, and it needs to be filled in the creation of the organisation.

Notes:

  • The organisations can be added (optionally) on the EthKudos directory. This information is only used to show an autocomplete on the landing page right now. But can be used for other purposes in the future (like show a list of all the organisations).
  • The organisation are Token ERC20, but have a limitation by design: is not possible to transfer kudos, the kudos just can be earned as part of a gratitude from other members.
Can I see my organisation on Etherscan? add

First of all, Etherscan is a website that shows the details of all the transactions, accounts and Smart contracts on Ethereum.

So yes, you are able to check all the transactions done on EthKudos in Etherscan.

Is my organisation like Bitcoin or Ethereum? add

No, your organisation is a Token ERC20, but it has the trading features limited, so nobody can send or receive it. So each token won't have more value than the prestige among the members a company, team or organisation.

Technical details

How is EthKudos built? add

EthKudos have 2 parts:

  • Ethereum: EthKudos has deployed some Smart contracts (like applications) on Ethereum, that provide the organisation creation and all the features that it needs to work (like member addition, polling creation, send gratitudes on a polling...).
  • Web dapp: is a web application that doesn't need its own server to store the data, just use the data provided by the Ethereum network, that store it safely.
How is EthKudos technically built? add

EthKudos need a lot of technologies to be developed and make it work.

  1. Ethereum side:
    • Solidity: is the programming language used by Ethereum.
    • Truffle: is the framework that helps to develop over Ethereum using Solidity. Provides a way to test the Smart contracts and deploy it (with some good practices, like incremental migrations).
    • Infura: provides an interface to connect to Ethereum. Used to deploy the Smart contracts.
  2. Web dapp side:
    • Angular: is the framework used to build all the application, it provides routing, services injection, template rendering...
    • Web3 (Ethereum API): is the JavaScript API to interact with Ethereum Network.
    • Truffle contract: makes the interaction with Smart contracts easier.
  3. Infrastructure:
    • CircleCI: provides a continuous integration, running tests, code quality and building the application to be deployed.
    • Heroku: provides a hosting for the web dapp, being easy to be deployed by CircleCI.
    • Cloudflare: is a middleware layer that adds https, cache, DDoS prevention and more features to the web dapp.
Can I check the code? add

Yes, all the code is on GitHub and anyone can contribute.

How many Smart contracts does EthKudos need to work? add

EthKudos needs some Smart contracts to work, there are:

  • KudosPoll: manage and stores a polling, the members of an organisation votes to others directly on this Smart contract. It stores all the kudos (with its messages).
  • KudosPollFactory: provides a middleware to create a KudosPoll instance, allowing to be upgraded the Smart contract.
  • KudosToken: manage an organisation, it stores the members, allow to create and close polls and stores the total number of kudos by the member.
  • KudosTokenFactory: provides a middleware to create a KudosToken instance, allowing to be upgraded the Smart contract.
  • KudosOrganisations: is responsible of the creation of new organisations (KudosToken) and store the address of the organisations that are in the directory.
  • KudosRouter: this Smart contract stores the address and version of KudosTokenFactory and KudosPollFactory, it allows to upgrade the new organisations and polling events.
